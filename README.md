# Página de login

Projeto react tela de login conectado ao backend provedor JWT

## 🚀 Começando

Essas instruções permitirão que você obtenha uma cópia do projeto em operação na sua máquina local para fins de desenvolvimento e teste.


### 📋 Pré-requisitos

```
 node v18.14.2
```

### 🔧 Instalação

Para instalar:

```
npm install
```
Para executar:

```
npm start
```
